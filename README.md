# Academic AI Assistant

An intelligent AI-powered platform for academic guidance and document analysis.

## 🚀 Key Features

- **Document Analysis**: Upload and analyze PDF, DOCX, PPTX files with AI-powered insights
- **AI Chat**: Context-aware conversations using uploaded documents
- **Google Authentication**: Secure sign-in with Google accounts
- **Multiple Chat Sessions**: Create and manage separate conversation threads
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices

## 🛠️ Technology Stack

- **React 18** with TypeScript
- **Vite** for fast development and building
- **Google Gemini API** for AI capabilities
- **PDF.js, Mammoth.js, PPTX Parser** for document processing

## 🚀 Getting Started

1. Clone the repository and install dependencies:
```bash
git clone <repository-url>
cd academic-ai-assistant
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open `http://localhost:3000` in your browser
